Chapter 29 — Multiple Linear Regression II: Inferences on the Model

Topics covered: distributions of SSE and SSR, regression ANOVA, overall F tests, \(R^2\)-to-F conversion, coefficient variance-covariance matrices, T tests, confidence intervals for model parameters, confidence intervals for mean response, prediction intervals for new observations, partial F tests for nested models, contrasts, and final-style model interpretation. The main exam traps are using the wrong degrees of freedom, treating overall significance as proof that every coefficient is significant, ignoring covariance terms in contrasts, confusing mean intervals with prediction intervals, and using individual T tests for joint hypotheses.
