Chapter 28 — Multiple Linear Regression I: Basic Model

Topics covered: model matrices, normal equations, least-squares estimator \(\mathbf b=(X^TX)^{-1}X^T\mathbf y\), fitted values, residuals, the hat matrix, SSE/SSR/SST decomposition, \(R^2\), polynomial regression as a linear model, dummy-variable designs, rank and collinearity, and projection geometry. The main exam traps are losing the intercept column, using \(n-p\) instead of \(n-p-1\), treating polynomial regression as nonlinear in the parameters, trusting \(R^2\) without checking residuals, and ignoring singular \(X^TX\).
