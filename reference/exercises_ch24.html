Chapter 24 — Non-Parametric Comparisons; Paired Tests and Correlation

Topics covered: Wilcoxon rank-sum tests for two independent samples, ties and large-sample caveats, paired T-tests, Wilcoxon signed-rank tests on paired differences, pairing versus pooling, tests for a correlation coefficient, Fisher transformation confidence intervals, assumptions, and final-style error diagnosis. The main exam traps are pairing independent samples, pooling genuinely paired samples, using a normal approximation with severe ties without caveat, using the wrong degrees of freedom, and forgetting to back-transform Fisher intervals.
