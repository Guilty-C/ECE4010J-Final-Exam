Chapter 19 — Single Sample Tests for the Mean and Variance

Topics covered: hypotheses, test statistics, rejection regions, P-values, known-variance Z tests for a mean, unknown-variance T tests for a mean, chi-squared tests for variance and standard deviation, one-sided and two-sided alternatives, assumptions, critical values, power and sample-size planning, and diagnose-the-error final-style problems. The main exam traps are using \(s\) with a Z critical value, using \(s/n\) instead of \(s/\sqrt n\), reversing one-sided alternatives, doubling a one-sided P-value incorrectly, forgetting \(n-1\) degrees of freedom, and applying chi-squared variance tests without normality.
